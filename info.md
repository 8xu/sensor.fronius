### Features:
* Pulls data from one or more Fronius inverters
* Creates sensors automatically or based on MONITORED_CONDITIONS
* Simple default configuration

### Minimal Configuration
```
sensor:
- platform: fronius
    ip_address: LOCAL_IP_FOR_FRONIUS
```
